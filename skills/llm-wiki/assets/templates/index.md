# インデックス

wiki の最初の discovery surface として使います。目的別入口は代表的 reader task への shortcut です。現役ページ一覧は Active Page Catalog です。active canonical durable page は現役ページ一覧に 1 回だけ載せ、1 行 summary と主要検索語を付けます。draft note、rejected note、deferred note、merged / archived duplicate は active page として載せません。

## 目的別入口

root の性質に合わせてカテゴリを変更してよい。全 page をここへ分類せず、読者がよく使う代表入口だけを置く。検索語は現役ページ一覧側に集約する。
